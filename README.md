# 📊 Projeto de Business Intelligence – Retail Business Intelligence Dataset

## 📌 Sobre o Projeto

Este projeto aplica conceitos de **Business Intelligence (BI)** para analisar **dados de vendas no varejo** com foco em transações de lojas de varejo.  
A base utilizada foi o **Retail Business Intelligence Dataset**, obtida no **Kaggle**, que contém registros de vendas, produtos, campanhas e atributos relevantes para análises comerciais.

O objetivo é transformar dados brutos em **insights estratégicos**, por meio de **limpeza, tratamento, modelagem e visualização de dados no Power BI**, culminando na criação de **quatro dashboards interativos** com diferentes perspectivas de análise.
